# Zcash Python test vectors

Code to generate test vectors for various parts of Zcash.

Requirements:
- `pyblake2`
- `chacha20poly1305` (for `sapling_note_encryption.py`).
- `numpy` (for `orchard_poseidon.py` and dependents).
